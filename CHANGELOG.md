# Changelog

## 0.2.0 - 2026-03-30

- Added signed cookie-based authentication with email/password, Google/GitHub OAuth readiness, and anonymous guest sessions.
- Added GDPR/privacy center with consent controls, data subject request intake, and audit-friendly storage models.
- Expanded monetization with hosted Stripe checkout for card/Klarna and manual verified request flows for Swish, BTC, XMR, LTC, and cash.
- Replaced the placeholder admin page with moderation, legal review, payment, privacy, and beta feedback queues.
- Added locale switching foundation for Swedish and English in the shell and operational pages.
- Added beta feedback collection, API documentation page, terms page, release documentation, and CI/CD automation.

## 0.1.0 - 2026-03-30

- Initial hosted-first MVP with Next.js, Prisma, Stripe foundation, Swedish feature pages, legal framing, and public API endpoints.
